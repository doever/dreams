### 一、基本面分析（公司内在价值）

你可以问：

- **这家公司是做什么的？它的主营业务和产品是什么？**
- **公司的盈利情况如何？净利润和营收是否持续增长？**
- **毛利率、净利率是多少？同行业比较怎么样？**
- **公司的资产负债情况如何？有没有高负债风险？**
- **公司的管理层是谁？口碑和历史表现怎么样？**
- **是否有护城河？技术壁垒、品牌力、专利情况如何？**

------

### 二、估值分析（股价是否被高估或低估）

你可以问：

- **当前的市盈率（PE）、市净率（PB）是多少？和历史比高还是低？**
- **与同行业平均估值相比，这只股票贵吗？**
- **有没有折价的合理理由（如行业周期、法律风险）？**

------

### 三、行业分析（所处行业的走势）

你可以问：

- **这家公司所在的行业目前是增长期、成熟期还是衰退期？**
- **行业的竞争格局怎么样？龙头企业是谁？**
- **政策利好还是打压？有无周期性风险？**
- **有哪些上下游关键企业或依赖？**

------

### 四、技术面分析（市场行为）

你可以问：

- **这只股票的趋势怎么样？处于上涨还是下跌阶段？**
- **关键的技术指标如均线、MACD、RSI 是什么信号？**
- **有没有突破重要支撑/阻力位？**
- **筹码集中度如何？是否有主力资金介入迹象？**

------

### 五、资金面与情绪面

你可以问：

- **近期有无机构增持、外资买入？北向资金动向如何？**
- **市场对这只股票的情绪是看涨还是看空？**
- **最近有没有利好消息或利空传闻？**
- **龙虎榜、大单成交情况如何？**

------

### 六、财务与分红政策

你可以问：

- **公司分红大方吗？历史上分红率如何？**
- **有没有股票回购计划？**
- **未来是否存在盈利释放的潜力？**

------

### 七、风险分析

你可以问：

- **有哪些潜在风险因素？政策风险、财务风险、行业变革？**
- **有没有官司、监管处罚或负面新闻？**
- **如果公司业绩不达预期，会造成多大影响？**

------

### 八、未来预期与催化剂

你可以问：

- **公司是否发布了新的发展规划或战略？**
- **有没有明确的成长催化剂，如新品发布、并购、业绩爆发？**
- **未来一两个季度的业绩指引怎么样？**





## 一、构建评分模型的步骤

### 1. **确定评分维度（指标）**

我们可以采用你前面提到的七八个方面，每个方面下设若干具体问题作为**评分项**。推荐结构如下：

| 维度           | 权重建议 | 子项数量 | 示例问题                               |
| -------------- | -------- | -------- | -------------------------------------- |
| 基本面         | 30%      | 3-5      | 营收增长是否稳定？毛利率是否领先行业？ |
| 行业前景       | 15%      | 2-3      | 行业是否处于成长期？是否政策支持？     |
| 财务安全       | 15%      | 2-3      | 负债率是否合理？现金流充足吗？         |
| 估值           | 15%      | 2-3      | 当前PE是否合理？是否处于历史低位？     |
| 技术面         | 10%      | 2        | 是否处于上升趋势？有无放量上涨？       |
| 资金动向       | 10%      | 2        | 机构/外资是否增持？主力是否入场？      |
| 风险与不确定性 | -15%     | 2        | 是否有重大诉讼或财务造假风险？         |



> 权重你可根据风格微调：如价值投资者重基本面与估值；投机者更看重技术面与资金动向。

------

### 2. **设置每项评分标准（建议5分制）**

| 得分 | 含义                     |
| ---- | ------------------------ |
| 5    | 极优：明显强于行业/历史  |
| 4    | 较优：小幅领先           |
| 3    | 一般：与行业大体持平     |
| 2    | 偏弱：存在劣势           |
| 1    | 很差：明显问题或重大隐患 |



> 对于**负面项**（如风险事件），可倒扣或设置负分权重。

------

### 3. **计算总分**

每项得分 × 权重后加总，最后得出一个0～100分的综合分值：

总分=∑i(得分i×权重i)\text{总分} = \sum_i (\text{得分}_i \times \text{权重}_i)总分=i∑(得分i×权重i)

------

## 🔧 实操建议

### A. 简单做法（初学者）：

- 用Excel建个打分表
- 每列一个问题，旁边手动填分，自动汇总加权

### B. 高级做法（自动化）：

- 把数据源接入 Python（如用 `yfinance`, `tushare`, `eastmoney`, `聚宽` 等）
- 写一套规则，自动打分和可视化

------

## ✅ 最后建议

- 不必追求完美权重，关键是**保持一致性和对比性**（比较不同股票时更有价值）
- 有时间可以建立自己的“股票池”，打完分排序，优先研究高分股票
- 打分模型是工具，不是预测器，要结合市场情绪和你对宏观、行业的判断
